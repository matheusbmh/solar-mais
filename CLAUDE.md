# Solar Mais — Contexto do Projeto

## Sobre o projeto
Site de captação de clientes para energia solar distribuída na Bahia.

- **URL publicada:** https://solarmais.surge.sh/
- **Deploy:** Surge (email: matheus.bragah@gmail.com)
- **Arquivos:** `index.html`, `logo.png`

## O que o site faz
Plataforma para conectar clientes COELBA ao modelo de geração distribuída, com desconto garantido de 20% na conta de luz sem investimento inicial. Inclui simulador de economia, mapa de irradiação solar e formulário de contato.

## Como fazer deploy após alterações
```
surge
```
Rodar dentro desta pasta. Vai pedir email/senha do Surge na primeira vez.
